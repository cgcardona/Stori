The TellUrStori DAW main window is where you work on your projects. You can access all of the major working areas of TellUrStori DAW in the main window.

![TellUrStori DAW Main Window](../assets/screenshots/main-window-overview.png)

## üéØ Main Interface Areas

### Control Bar
The **Control Bar** runs across the top of the main window and includes:
- **Project Controls** - New, Open, Save project buttons
- **Transport Controls** - Play, Pause, Stop, Record, and navigation
- **Time Display** - Current playback position and project tempo
- **Master Volume** - Overall project volume control
- **CPU Meter** - Real-time processing load indicator

### Tracks Area
The central part of the main window is the **Tracks Area**. You record and arrange the musical material in your project on the tracks in the Tracks area. There are **audio tracks** for recording and playing back audio content. When you generate AI music or import an audio file, it appears as a rectangular **region** on the selected track.

### Track Headers
On the left side of each track, you'll find the **Track Headers** containing:
- **Track Number** - Sequential track numbering
- **Track Name** - Editable track names (double-click to edit)
- **Record Button** - Red circle (‚óè) to arm track for recording
- **Mute Button** - (M) to silence the track
- **Solo Button** - (S) to isolate the track for playback
- **Volume Slider** - Horizontal slider for track level
- **Pan Knob** - Stereo positioning control
- **AI Generate Button** - Create AI music directly on the track

### Mixer Panel
The **Mixer Panel** on the right provides detailed mixing controls:
- **Channel Strips** - Complete mixing controls for each track
- **EQ Section** - 3-band equalizer (High, Mid, Low)
- **Effects Slots** - Professional audio effects processing
- **Level Meters** - Real-time audio level monitoring
- **Bus Routing** - Auxiliary send controls for effects

### Timeline Ruler
Across the top of the Tracks area is the **Timeline Ruler**, which shows:
- **Time Markers** - Precise time positions (minutes:seconds)
- **Beat Divisions** - Musical timing grid
- **Playhead** - Red vertical line showing current position
- **Cycle Region** - Yellow highlighted loop area (when enabled)

## üîß Customization

### Editable Parameters
Most parameters in TellUrStori DAW support **double-click editing**:
- **Double-click** any parameter to enter edit mode
- **Type** the desired value with appropriate units
- **Press Enter** to save or **Escape** to cancel
- **Units are automatic** - Hz for frequencies, dB for levels, % for percentages

- **Space** - Play/Pause
- **R** - Record
- **C** - Toggle Cycle Mode
- **‚åòM** - Show/Hide Mixer
- **‚åòDelete** - Delete Selected Track
