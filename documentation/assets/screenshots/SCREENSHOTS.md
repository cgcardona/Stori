## üìù Screenshot Guidelines

### Technical Requirements
- **Resolution**: Retina quality (2x scaling)
- **Format**: PNG with transparency where appropriate
- **Compression**: Optimized for web viewing
- **Annotations**: Use callouts and labels for UI elements

### Content Guidelines
- **Clean Interface**: Remove personal data and use placeholder content
- **Consistent Theme**: Use the blue-purple-pink gradient theme
- **Professional Quality**: Ensure crisp, clear visibility of all elements
- **Realistic Data**: Use meaningful project names and track content

### Naming Convention
- Use descriptive, kebab-case names
- Include interface area in filename
- Version with date if multiple iterations needed
