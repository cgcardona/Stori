## üèóÔ∏è Professional DAW Terminology

### **Standard DAW Terms**
- **Arrangement View** - Main timeline area for composing
- **Channel Strip** - Complete set of mixer controls for one track
- **Playhead/Cursor** - Position indicator showing current time
- **Track Lane** - Horizontal area where audio/MIDI regions are placed
- **Session** - Complete project with all tracks and settings
- **Regions/Clips** - Audio or MIDI segments on tracks
- **Fader** - Volume control slider
- **Knob/Pot** - Rotary control for parameters
- **Bus** - Signal routing path for effects processing
- **Send** - Route signal to effects processor
- **Return** - Bring processed signal back to mix

### **TellUrStori-Specific Terms**
- **STEM** - Individual track element that can be tokenized as NFT
- **AI Region** - Audio content generated by MusicGen AI
- **Blockchain Track** - Track prepared for NFT minting
- **Marketplace STEM** - NFT-tokenized audio available for trading
- **Generation Template** - AI prompt configuration for music creation

This terminology reference helps maintain consistency when discussing UI elements, writing documentation, or communicating about interface changes.
